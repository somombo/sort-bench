How a curve is computed

For every generated array, the minimum time over repeated runs is kept — stripping OS jitter, GC pauses and context switches. Across the independent random arrays at each axis value, each point is the median of those minima. Only the sort itself is timed; I/O, parsing and allocation are excluded.

Switch Per-array reduction to Warm-up to keep only the last repetition of each array instead of the minimum — emulating warm-up benchmarking, where the final rep is the warmed-up measurement.
